## 开始使用
* 请允许AppStoreMonitor显示通知
* 输入您的App对应的AppID，点击开始
* AppStoreMonitor会每隔1分钟获取一次应用最新版本信息
* 若检测到新版本，可通过浏览器推送通知您
* 再也无需反复打开AppStore查看应用更新是否生效
* 仅限AppStore的应用
